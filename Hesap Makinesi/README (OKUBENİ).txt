English:
This application not only performs operations such as addition, subtraction, multiplication, division, modulus, exponentiation, and square root on two entered numbers, but also has a special feature: if the result is 16, it directs the user to a password entry screen for a special safe. If the user enters the wrong password three times, they are logged out of the application. Entering the password 'metehan123' grants access to the special safe.
